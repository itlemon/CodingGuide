# 数据结构与算法阅读指南 <Badge text="Beta" />

## 什么是数据结构与算法？

:mag_right:学习数据结构与算法之前，我们要理解什么是数据结构与算法。

数据结构，就是一组数据的存储结构。

算法，就是操作数据的一组方法。

数据结构是为算法服务的，算法要作用在特定的数据结构之上。

基于上面的说法，我们基本可以说：`程序 = 数据结构 + 算法`。
